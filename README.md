# bill-front

这是一个基于vue的前端项目，主要实现了账单的添加、查询、统计功能。

## 技术栈

- axios ^0.18.0
- echarts ^4.2.0-rc.2
- element-ui ^2.4.11
- vue ^2.5.17
- vue-router ^3.0.2
- vuex ^3.0.1
